# Nehal Electrònic

Informational website for **Nehal Electrònic**, a mobile phone shop in  
La Bisbal d’Empordà.

## About
This website shows:
- Mobile repair services
- Accessories available in store (cases, chargers, headphones)
- Contact information and opening hours

This website does **not** sell products online.
